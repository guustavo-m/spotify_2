# Spotify 2

Um aplicativo mobile em React Native desenvolvido com Expo, inspirado na plataforma de streaming de música Spotify.

## Funcionalidades

- Aplicativo multiplataforma (iOS, Android e Web)
- Desenvolvido com React Native e Expo
- Interface moderna com tema claro

## Pré-requisitos

Antes de começar, certifique-se de ter instalado:

- Node.js (versão 18 ou superior)
- npm ou yarn
- Expo CLI
- Para desenvolvimento mobile:
  - Xcode (para desenvolvimento iOS no macOS)
  - Android Studio (para desenvolvimento Android)

## Instalação

1. Clone o repositório:

```bash
git clone <repository-url>
cd spotify_2
```

2. Instale as dependências:

```bash
npm install
```

3. Inicie o servidor de desenvolvimento do Expo:

```bash
npm start
```

## Executando o Aplicativo

### Em um Dispositivo Físico ou Emulador

- **iOS**: pressione `i` no terminal para abrir no Simulador iOS (apenas macOS)
- **Android**: pressione `a` no terminal para abrir no Emulador Android
- **Dispositivo Físico**: escaneie o QR Code com o aplicativo Expo Go

### Na Web

Pressione `w` no terminal para abrir no navegador padrão.
